# 新世纪五笔 For Rime
该方案是为 Rime 制件的新世纪五笔及新世纪五笔拼音方案和简入繁出。

## 输入法方案
输入法方案在 schemas 目录下。

### 新世纪五笔
新世纪五笔词库来源于原有新世纪五笔词库和五笔86版词库，外加 ibus-table-extraphase 中的词库

### 新世纪五笔拼音
拼音方案来自五笔拼音方案， 从 wubi_pinyi 修改而来。

### 新世纪五笔简入繁出
简入繁出方案修改自五笔86版简入繁出，替换为新世纪码表。

## 程序

### wubi.cpp
编码的一个小程序
### scel.py
解析词库的一个小程序
